## 1. 서비스 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | 무역통계 대시보드 (가칭) |
| 목적 | 한국무역협회 K-stat 기존 서비스를 대체하는 AI 연동 무역통계 시각화 웹 앱 |
| 핵심 가치 | 숫자만 나열된 기존 통계를 LLM과 결합해 "왜?"를 설명해주는 인사이트 서비스 |
| 주 사용자 | 무역회사 실무자, 대학교수, 연구진 (월 약 3~5만 명) |
| 플랫폼 | 반응형 웹 (PC 우선, 태블릿·모바일 지원) |
| 데이터 범위 | 2023.01 ~ 2026.02, 한국 기준 대세계 수출입 (추후 중·일·미 확장 가능) |

## 3. 데이터 명세

### 3-1. 핵심 데이터 (CSV, 무역협회 제공)

| 컬럼 | 설명 | 단위 |
|------|------|------|
| YYMM | 기준 년월 (YYYYMM) | — |
| CTR_NAME | 교역 상대국명 (한국어) | — |
| MTI_CD | MTI 3단위 품목 코드 | — |
| MTI_NAME | MTI 품목명 | — |
| EXP_AMT | 수출 금액 (FOB 기준) | USD |
| EXP_QTY | 수출 수량 | 개 |
| EXP_WGT | 수출 중량 | kg |
| IMP_AMT | 수입 금액 (CIF 기준) | USD |
| IMP_QTY | 수입 수량 | 개 |
| IMP_WGT | 수입 중량 | kg |

**규모:** 약 534,652행 / 250개국 / 198개 품목 (MTI 3단위) / 38개월

### 3-2. 데이터 품질 주의사항

- **IMP_AMT = 0인 행 42.3%:** "수입 없음"이 아닌 경우 포함. 필터 시 0 vs null 구분 필요
- **수출 FOB / 수입 CIF:** 동일 품목도 수입 금액이 높게 나올 수 있음. 무역수지 표시 시 사용자에게 안내
- **2026년 데이터:** 1~2월만 존재. 연도별 비교 시 "불완전 연도" 레이블 표시
- **홍콩:** 중계무역 허브. 실제 최종 목적지와 다를 수 있음 → LLM 시스템 프롬프트에 명시
- **라이베리아·마셜제도:** 선박 등록지. 실제 교역 상대국 아님 → LLM 시스템 프롬프트에 명시
- 중계무역, 선박 등록지에 상관없이 CSV 파일에 표기된 내용 그대로 표시

## 4. 전체 페이지 구조

```
P1 — 랜딩 (탭 두 개짜리 단일 화면)
  │
  ├─ [국가별 탭]
  │    └─ TOP30 국가 클릭 ──── P2-A 국가별 상세
  │                                 └─ 돌아가기 → P1 국가별 탭
  │
  └─ [품목별 탭]
       └─ TOP3 품목 "상세 보기" 클릭 ─── P2-B 품목별 상세
                                          └─ 돌아가기 → P1 품목별 탭
```

> **라우팅 원칙:** 돌아가기는 항상 진입 전 탭(국가별/품목별)으로 복귀.  
> 국가별 탭에서 P2-A 진입 후 상단 탭을 품목별로 변경하면 → P1 품목별 탭으로 이동 (랜딩 국가별로 돌아가지 않음).

**URL 파라미터 규칙:**

| 페이지 | URL 예시 |
|--------|---------|
| 랜딩 (국가별 탭) | `/?tab=country` |
| 랜딩 (품목별 탭) | `/?tab=product` |
| 국가별 상세 | `/?tab=country&country=중국&year=2025&period=annual&mode=export` |
| 품목별 상세 | `/?tab=product&year=2025&period=annual&mode=export` |

## 5. 공통 UI 컴포넌트

### 5-1. Split Panel 레이아웃

```
┌──────────────────────────────────────┬─────────────┐
│  대시보드 영역 (75%)                  │  챗봇 (25%) │
└──────────────────────────────────────┴─────────────┘
```

- 챗봇 닫기 → 대시보드 100% 확장 (오버레이 아님)
- 모바일: 챗봇을 하단 슬라이드업 패널로 전환
- 비율은 75:25 기준, 지도·트리맵 컷오프 여부 확인 후 조정

### 5-2. 필터 바 (공통)

| 필터 | 옵션 | 기본값 |
|------|------|--------|
| **수출/수입 토글** | 수출 / 수입 | 수출 |
| 연도 | 2023 / 2024 / 2025 / 2026 | 최신 연도 |
| 월 | 전체 / 1~12월 | 전체 |
| 금액조회 | 해당월 / 누적 / 연간 | 연간 |

> **수출/수입 토글은 전역 상태.** 토글 변경 시 모든 지도·차트·KPI·트리맵 데이터가 해당 모드 기준으로 일괄 갱신됨.

### 5-3. KPI 카드 바 (공통)

**수출 모드:**

| 카드 | 내용 | 비고 |
|------|------|------|
| 총 수출액 | $XXX억 + 증감률 | 필터 연동 |
| 총 수입액 | $XXX억 + 증감률 | 필터 연동 |
| 무역수지 | $XXX억 | 양수=파랑(흑자), 음수=빨강(적자) |

**수입 모드:**

| 카드 | 내용 | 비고 |
|------|------|------|
| 총 수입액 | $XXX억 + 증감률 | 필터 연동 (강조) |
| 총 수출액 | $XXX억 + 증감률 | 필터 연동 |
| 무역수지 | $XXX억 | 항상 표시, 양수=파랑, 음수=빨강 |

> 무역수지 카드는 수출/수입 모드 모두에서 항상 표시.

**증감률 레이블 동적 변경:**
- 해당월 → "전월 대비"
- 연간 → "전년 대비"
- 누적 → "전기 누적 대비"

### 5-4. 교역국 필터 — 대륙별 드롭다운

- 대륙명 클릭 = 대륙 전체 선택 / 화살표 클릭 = 하위 국가 펼치기 (독립 동작)
- 국가명 클릭 = 해당 국가 선택
- 선택 상태는 상단 chip으로 표시
- 수출 모드: "수출 상대국" 기준 TOP30 / 수입 모드: "수입 상대국(공급국)" 기준 TOP30
- 사용자가 선택한 연도·월·금액조회 기준으로 해당 시기의 금액 순 TOP30 표시

### 5-5. 품목 필터 — MTI 대분류

- 대분류명 클릭 = 대분류 선택
- 랜딩 품목별 탭에서 품목 필터 클릭 시 대분류만 드릴다운 형식으로 표시

## 6. P1 — 랜딩 페이지 (국가별 탭)

### 6-1. 목적

한국의 수출입 현황을 지도 위에서 직관적으로 파악하고, 관심 국가 클릭 시 상세 분석으로 진입

### 6-2. 레이아웃

```
┌─────────────────────────────────────────────────────┐
│  [탭] 국가별 (활성) | 품목별                          │
├─────────────────────────────────────────────────────┤
│  [필터 바]  수출/수입 | 연도 | 월 | 금액조회           │
├─────────────────────────────────────────────────────┤
│  [KPI]  총 수출 | 총 수입 | 무역수지                   │
├─────────────────────────────────────────────────────┤
│                                                     │
│  [코로플레스 지도 — Leaflet.js]                       │
│  국가별 수출액(또는 수입액) 색상 강도 표현             │
│                                    [+]              │
│                                    [-]  ← 줌 버튼   │
├─────────────────────────────────────────────────────┤
│  [색상 범례 스펙트럼]  1~3위 ████████░░░ 22~30위     │
└─────────────────────────────────────────────────────┘
```

### 6-3. 지도 인터랙션

| 동작 | TOP30 국가 | TOP30 외 국가 |
|------|-----------|--------------|
| 호버 | 툴팁: 국가명 + 수출입액 + 순위 / **cursor: pointer (손가락)** | 툴팁: 국가명 + "상세 데이터 제한" 안내 / cursor: default |
| 클릭 | P2-A 국가별 상세로 이동 | 반응 없음 |
| 색상 | 수출(또는 수입)액 비례 파랑 계열 | #B5D4F4 |

> **cursor 처리:** TOP30 국가에만 `cursor: pointer` 적용. TOP30 외 국가 및 바다 영역은 `cursor: default`.

### 6-4. 색상 구간 (수출 모드 / 수입 모드 동일 팔레트)

| 수출(수입)액 기준 국가 순위 | 색상 | 의미 |
|-----------|------|------|
| 1 ~ 3위 | #042C53 | 최대 교역국 |
| 4 ~ 9위 | #0C447C | |
| 10 ~ 15위 | #185FA5 | |
| 16 ~ 21위 | #378ADD | |
| 22 ~ 30위 | #85B7EB | |
| TOP30 외 | #B5D4F4 | 데이터 있음, 상세 미제공 |
| 바다 | #D3D1C7 | 옅은 회색 |

> 수입 모드 전환 시 동일 팔레트 적용, 수입액 기준 TOP30 재계산.

### 6-5. 색상 범례 스펙트럼 (지도 하단 고정)

```
[●] 1~3위   [●] 4~9위   [●] 10~15위   [●] 16~21위   [●] 22~30위   [●] TOP30 외
 #042C53    #0C447C     #185FA5       #378ADD       #85B7EB       #B5D4F4
```

- 지도 하단에 가로 스펙트럼 바로 표시
- 수출 모드: "수출액 기준" 레이블 / 수입 모드: "수입액 기준" 레이블 자동 변경

### 6-6. 줌 컨트롤 (네이버 지도 스타일)

- 위치: 지도 우측 하단 고정
- 구성: `[+]` 버튼 (줌인) / `[-]` 버튼 (줌아웃)
- 스타일: 흰색 배경, 테두리 포함, 둥근 모서리, 그림자
- 기존 Leaflet 기본 줌 컨트롤 비활성화 후 커스텀 버튼으로 대체

### 6-7. TOP30 기준

- 선택된 필터 기간 기준으로 수출(또는 수입)액 TOP30 실시간 산출
- 필터 또는 수출/수입 토글 변경 시 TOP30 재계산 → 색상 및 클릭 가능 여부 업데이트

### 6-8. 지도 구현 주의사항

- GeoJSON 국가 경계: Natural Earth 데이터 사용
- 국가명 매핑: GeoJSON 영문명 ↔ CTR_NAME 한국어 매핑 테이블 별도 작성 (250개국)
- 국가명 영문·한글 일치 여부 확인 후 매핑 테이블 작성

## 7. P1 — 랜딩 페이지 (품목별 탭)

### 7-1. 목적

한국의 전체 수출입 품목 구성을 소분류 기준 트리맵으로 시각화하고, 관심 품목의 상세 분석으로 진입

### 7-2. 레이아웃

```
┌──────────────────────────────────────────────────┐
│  [탭] 국가별 | 품목별 (활성)                       │
├──────────────────────────────────────────────────┤
│  [필터 바]  수출/수입 | 연도 | 월 | 금액조회 | 품목  │
├──────────────────────────────────────────────────┤
│  [KPI]  총 수출 | 총 수입 | 무역수지               │
├──────────────────────────────────────────────────┤
│                                                  │
│  [소분류 기준 트리맵 — D3.js]                     │
│  대분류 색상 구분 / 소분류(198개) 면적 표현         │
│                                                  │
├──────────────────────────────────────────────────┤
│  [대분류 아이콘 필터 10개]                         │
└──────────────────────────────────────────────────┘
```

### 7-3. 트리맵 구성 및 렌더링 규칙

- **면적 기준:** MTI 소분류(198개) 기준 수출(또는 수입)액 비례
- **색상 구분:** MTI 대분류(10개) 기준 색상 부여
- **텍스트 처리 (v2.0 수정):**
  - 품목명은 한국어(MTI_NAME)로 표시
  - 박스 크기보다 텍스트가 길면 자동 줄바꿈 (`word-break: keep-all`)
  - 텍스트는 반드시 박스 안에서만 표시 (`overflow: hidden`)
  - 박스가 너무 작아 텍스트 표시가 불가능한 경우 텍스트 숨김 처리
- **Root 노드 배경 (v2.0 수정):** 트리맵 하단 Root 합산 배경 표시 안 함. 빈 배경 유지.

### 8-2. 레이아웃

**수출 모드:**

| 카드 | 내용 | 연동 |
|------|------|------|
| ← 돌아가기 | P1 국가별 탭 복귀 | — |
| 지역 | 대륙명 | 고정 |

**수입 모드 (레이블 변경):**

| 카드 | 내용 | 연동 |
|------|------|------|
| ← 돌아가기 | P1 국가별 탭 복귀 | — |
| 지역 | 대륙명 | 고정 |
| 국가명 | 코드 + 국가명 | 선택 국가 |
| 수입 순위 | 전체 수입 기준 국가 순위 | 필터 연동 |
| 수입 비중 | 전체 수입 대비 % | 필터 연동 |

### 8-4. 탭 1 — 품목별 트리맵

**수출 모드:**
- 시각화: 해당 국가 수출 품목 구성 트리맵 (중분류 MTI 2단위 기준)
- 트리맵 색상: 대분류 색상 체계 동일 적용
- 호버 툴팁: 품목명 / **수출 점유율** (해당 국가 수출 내 비중) / **수출 성장률** (전년 동기 대비) / **수출 금액**
- 클릭: 이동 없음. 호버 툴팁만 표시.
- "상세 페이지로 이동" 버튼 표시 안 함
- 텍스트 처리: 박스 내 자동 줄바꿈, overflow hidden, 한국어 표시
- 필터 연동: 연도·월·금액조회 변경 시 데이터 갱신

**수입 모드 (레이블만 변경, 구조 동일):**
- 시각화: 해당 국가 수입 품목 구성 트리맵 (중분류 기준)
- 호버 툴팁: 품목명 / **수입 점유율** / **수입 성장률** / **수입 금액**

### 8-5. 탭 2 — 시계열 추이

- 월별 수출액 / 수입액 / 무역수지 라인 차트 (수출/수입 모드 무관하게 항상 3개 라인 모두 표시)
- Y축 좌: 수출·수입액 / Y축 우: 무역수지
- 라인: 수출(파랑 실선), 수입(주황 점선), 무역수지(초록 실선)
- 시계열 추이 상단에 **연도 셀렉터** 별도 표시. 선택한 해당 연도의 1~12월 데이터 표시
- X축: 해당 연도의 월별 (1월~12월)
- 빈 월(데이터 없음): 빈값 처리

### 8-6. 챗봇

초기 메시지 — 수출 모드: `"[국가명]과의 수출 현황입니다. 특정 품목에 대해 질문해주세요."`  
초기 메시지 — 수입 모드: `"[국가명]으로부터의 수입 현황입니다. 특정 품목에 대해 질문해주세요."`

LLM 컨텍스트:
- 현재 국가 + 필터 상태 + 수출/수입 모드
- 상위 5개 품목 + 수출입 금액 + 비중
- 전기 대비 증감 TOP3
- 해당 시기 거시경제 데이터 (환율, 기준금리)
- 도메인 지식 시스템 프롬프트

### 9-3. 좌측 카드 — 고정값 vs 동적값

**수출 모드:**

| 카드 | 값 종류 | 설명 |
|------|---------|------|
| 품목명 | **동적** | 드릴다운 단계에 따라 변경 |
| 전월 수출액 | **고정** | 데이터셋 최신 실월 기준. 필터 변경과 무관 |
| 수출 증감률 | **고정** | 데이터셋 최신 실월 기준. 필터 변경과 무관 |

**수입 모드 (레이블 변경, 구조 동일):**

| 카드 | 값 종류 | 설명 |
|------|---------|------|
| 품목명 | **동적** | 드릴다운 단계에 따라 변경 |
| 전월 수입액 | **고정** | 데이터셋 최신 실월 기준. 필터 변경과 무관 |
| 수입 증감률 | **고정** | 데이터셋 최신 실월 기준. 필터 변경과 무관 |

> **개발 핵심:** 전월 수출입액·증감률은 필터 API와 분리된 별도 캐싱 값. 필터 변경 시 재요청하지 않음.

### 9-4. 품목 드릴다운 — 2단계 구조

**단계 1 — 대분류 (초기 상태)**

- 트리맵: 10개 MTI 대분류를 수출(수입)액 비례 면적으로 표시
- 좌측 카드 품목명: "전체"
- 브레드크럼: 없음 (최상위)
- 텍스트 처리: 박스 내 자동 줄바꿈, overflow hidden, 한국어 표시
- Root 노드 배경: 표시 안 함 (빈 배경)
- 박스 클릭 → 단계 2로 이동

**단계 2 — 중분류 (최하단)**

- 트리맵: 선택 대분류 내 중분류를 수출(수입)액 비례 면적으로 표시
- 좌측 카드 품목명: 선택 대분류명으로 변경
- 브레드크럼 (시각화 영역 상단): `전체 › [대분류명]`
- 전월 수출(수입)액·증감률: 선택 대분류 기준 고정값으로 교체
- 텍스트 처리: 박스 내 자동 줄바꿈, overflow hidden, 한국어 표시
- Root 노드 배경: 표시 안 함
- 돌아가기 클릭 → 단계 1로 복귀

### 9-6. 탭 2 — 수출(수입)액 추이

### 9-9. 챗봇

## 10. LLM 연동 설계

### 10-1. 시스템 프롬프트 (고정)

```
당신은 한국 무역통계 분석 어시스턴트입니다.

[도메인 지식]
- 수출 금액은 FOB(선적 기준), 수입 금액은 CIF(보험·운임 포함) 기준입니다.
  동일 품목도 수입 금액이 더 높게 나올 수 있습니다.
- 홍콩은 중계무역 허브입니다. 한국의 홍콩 수출 중 상당 부분이 중국으로 재수출됩니다.
- 라이베리아·마셜제도는 실제 교역 상대국이 아닌 선박 등록지입니다.
- MTI 코드 831은 반도체, 741은 자동차, 133은 석유제품입니다.
- 2023년 한국 무역 적자(-10B), 2024~2025년 흑자 전환(+52B, +77B).

[답변 원칙]
- 숫자 관련 질의는 반드시 집계 결과 기반으로 답변. 추측 금지.
- 인과관계 단정 금지. "~의 영향이 컸던 것으로 보입니다" 표현 사용.
- 데이터 범위(2023.01~2026.02) 밖 질문 시 명확히 안내.
- 거시경제 데이터와 수출입 데이터를 연계해 맥락 있는 해석 제공.
- 수입 질의 시 CIF 기준으로 실제 교역 단가보다 높을 수 있음을 안내.
```

```json
    "country": null,
    "year": 2025,
    "month": null,
    "period": "annual"
  },
  "aggregated_data": {
    "top5_items": [],

  }
}
```

### 10-3. 가드레일

- 숫자 답변은 반드시 `aggregated_data` 기반
- 환율 없이 원화 환산 금지
- "확실히", "반드시" 등 단정 표현 금지
- 2026년 3월 이후 데이터 질의 시 범위 초과 안내

## 12. Supabase DB 설계 주의사항

- **복합 인덱스:** `(YYMM, CTR_NAME, MTI_CD)` — 필터 조합 쿼리 최적화
- **집계 캐싱:** 전월 수출입액·증감률 등 고정값은 별도 캐싱 테이블로 관리
- **대분류 집계:** `LEFT(MTI_CD::text, 1)` 기준 GROUP BY
- **중분류 집계:** `LEFT(MTI_CD::text, 2)` 기준 GROUP BY
- **수입 집계:** 동일 쿼리 구조, `EXP_AMT` → `IMP_AMT` 컬럼 전환
- **IMP_AMT = 0 처리:** WHERE IMP_AMT > 0 조건 필수 (42.3% 0값 행 제외)
